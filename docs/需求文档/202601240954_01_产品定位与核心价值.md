## 产品名称

**Craft Creator** - 基于 AI Agent 的智能内容创作平台

## 产品定位

一个面向自媒体创作者的**智能学习型 AI Agent 助手**，以脚本为中心，打通从选题到发布的全流程创作工作流。不同于传统的"工具集合"，Craft Creator 是一个**会思考、会学习、会主动协助**的创作伙伴。

## 目标用户

**主要用户群体**：自媒体创作者（优先）
- 抖音、B站、小红书等平台的内容创作者
- 需要高效产出优质内容的个人创作者
- 小型创作工作室和 MCN 机构

**次要用户群体**：科技工作者（后续扩展）
- 科研人员、学者
- 需要协助研究、写论文的专业人士

## 核心价值主张

### 1. AI Agent 驱动的智能工作流
告诉 Agent 你的创作意图，它会主动完成选题研究、脚本撰写、素材收集、内容生成等复杂任务。不是简单的 AI 功能叠加，而是真正理解创作流程的智能助手。

**示例**：
```
用户：我想做一期关于 AI 绘画工具的评测视频
Agent：好的，我来帮你规划...
      [自动完成选题分析、脚本生成、素材推荐、视频制作]
```

### 2. 全流程数据打通
所有创作数据（灵感、脚本、素材、成品）互相关联，形成个人创作知识库。每次创作都会积累经验，供未来复用。

**数据流**：
```
选题灵感 → 脚本创作 → 素材匹配 → 内容生成 → 发布分析
    ↓          ↓          ↓          ↓          ↓
         [创作知识库：持续学习和优化]
```

### 3. 智能学习与个性化
AI 持续学习你的创作风格、偏好和成功模式，越用越懂你。

**学习内容**：
- 内容风格（节奏、语气、视觉风格）
- 素材偏好（配色、音乐、镜头语言）
- 成功模式（爆款内容的共同特征）

### 4. 三层素材管理体系

**素材来源层**：整合本地素材 + 全网资源
- 本地文件自动扫描和整理
- 全网素材平台聚合（Unsplash、Pexels 等）
- AI 素材生成（图片、音乐）

**素材管理层**：项目库管理
- 以项目为单位组织素材
- AI 自动分类和打标签
- 素材关系图谱

**素材使用层**：智能推荐
- 创作时根据脚本内容自动推荐
- 自然语言搜索素材
- 素材复用和优化建议

### 5. 一次创作，多端分发
脚本自动适配图文、短视频、长视频等多种形式，适配不同平台（抖音、小红书、B站等）。

**内容策略**：
```
脚本（核心）
  ├─→ 图文内容（小红书、公众号）
  ├─→ 短视频（抖音、快手）
  └─→ 长视频（B站、YouTube）
```

## 与现有工具的区别

### vs 剪映
- **剪映**：专注视频剪辑工具
- **Craft Creator**：覆盖创作全流程的 Agent 系统，从选题到发布

### vs Notion AI
- **Notion AI**：通用文档助手
- **Craft Creator**：深度理解内容创作领域的专业 Agent

### vs Canva
- **Canva**：设计工具
- **Craft Creator**：从创意到成品的完整解决方案，强调 AI 主动协作

## 核心差异化优势

1. **基于 AI Agent 的智能工作流**：不是工具，是会思考的助手
2. **全流程数据打通和智能复用**：建立个人创作知识库
3. **智能学习型系统**：越用越懂你
4. **跨平台内容适配和分发**：一次创作，多端分发

## 产品愿景

成为自媒体创作者的"数字分身"，让创作更高效、更有创意、更有价值。
