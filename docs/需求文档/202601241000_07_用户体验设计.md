## UI/UX 设计原则

### 1. 对话优先，工具辅助
**理念**：AI Agent 是主要交互方式，传统 UI 控件作为补充

**实现**：
- Agent 对话框始终可见（固定在界面中）
- 支持快捷键快速聚焦对话框（Cmd/Ctrl + /）
- 所有功能都可以通过对话完成
- 传统 UI 提供精确控制（给高级用户）

**示例**：
```
用户可以说："帮我创建一个关于 AI 的视频项目"
也可以点击："新建项目" 按钮
```

---

### 2. 上下文感知
**理念**：UI 根据当前创作阶段自动调整，减少认知负担

**实现**：
- 选题阶段：突出灵感和参考内容
- 脚本阶段：突出编辑器和结构视图
- 制作阶段：突出素材库和预览器
- 发布阶段：突出平台选择和发布选项

**状态切换**：
```
[选题] → [脚本] → [素材] → [生成] → [发布]
  ↓        ↓        ↓        ↓        ↓
灵感面板  编辑器   素材库   预览器   发布面板
```

---

### 3. 非侵入式 AI 建议
**理念**：AI 建议不打断创作流程，用户可以随时采纳或忽略

**实现**：
- AI 建议以侧边栏或浮窗形式展示
- 使用柔和的视觉提示（不使用弹窗）
- 一键采纳或忽略
- 可以关闭 AI 建议

**视觉设计**：
```
┌─────────────────────────────────────┐
│  脚本编辑器                          │
│                                     │
│  正文内容...                         │  ┌──────────────┐
│                                     │  │ 💡 AI 建议   │
│                                     │  │              │
│                                     │  │ 这段开场可以 │
│                                     │  │ 更吸引人...  │
│                                     │  │              │
│                                     │  │ [采纳] [忽略]│
│                                     │  └──────────────┘
└─────────────────────────────────────┘
```

---

### 4. 即时反馈
**理念**：所有操作都有即时的视觉反馈，让用户知道发生了什么

**实现**：
- 操作成功：绿色提示 + 动画
- 操作失败：红色提示 + 错误信息
- 处理中：进度条 + 状态文本
- 支持撤销/重做（Cmd/Ctrl + Z）

**反馈类型**：
```
✓ 成功：脚本已保存
✗ 错误：素材加载失败
⏳ 处理中：正在生成视频... 45%
```

## 关键交互流程

### 快速创作流程（5 步完成）

```
1. 新建项目
   ↓
   点击"新建项目"或按 Cmd+N
   ↓
   Agent 对话框弹出

2. 描述意图
   ↓
   告诉 Agent："我想做一期关于 AI 的视频"
   ↓
   Agent 询问细节（平台、时长、类型）

3. 生成脚本
   ↓
   Agent 自动生成脚本大纲
   ↓
   在编辑器中调整脚本

4. 匹配素材
   ↓
   Agent 实时推荐素材
   ↓
   拖拽素材到脚本段落

5. 生成内容
   ↓
   点击"生成内容"
   ↓
   选择平台和格式
   ↓
   预览并导出
```

### 素材搜索流程

```
1. 打开素材库
   ↓
   按 Cmd+M 或点击左侧"素材库"

2. 搜索素材
   ↓
   输入："找一张日落的照片"
   ↓
   AI 理解语义，返回相关素材

3. 预览素材
   ↓
   悬停显示大图
   ↓
   点击查看详情

4. 使用素材
   ↓
   拖拽到脚本编辑器
   ↓
   或点击"添加到项目"
```

## 动画和过渡

### 过渡时长
- 快速：`150ms`
- 标准：`300ms`
- 慢速：`500ms`

### 缓动函数
- 标准：`cubic-bezier(0.4, 0.0, 0.2, 1)`
- 进入：`cubic-bezier(0.0, 0.0, 0.2, 1)`
- 退出：`cubic-bezier(0.4, 0.0, 1, 1)`

### 常用动画
- 淡入淡出：`opacity` 过渡
- 滑入滑出：`transform: translateY()` 过渡
- 缩放：`transform: scale()` 过渡
- 旋转：`transform: rotate()` 过渡
