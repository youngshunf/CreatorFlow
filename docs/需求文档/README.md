## 产品概述

**Craft Creator** 是一个基于 AI Agent 的智能内容创作平台，面向自媒体创作者，以脚本为中心，打通从选题到发布的全流程创作工作流。

### 核心价值主张

1. **AI Agent 驱动的智能工作流**：不是工具，是会思考的助手
2. **全流程数据打通**：所有创作数据互相关联，形成知识库
3. **智能学习与个性化**：越用越懂你
4. **三层素材管理体系**：来源层 → 管理层 → 使用层
5. **一次创作，多端分发**：脚本自动适配多种形式和平台

### 目标用户

- **主要用户**：自媒体创作者（抖音、B站、小红书等平台）
- **次要用户**：科技工作者（后续扩展）
