# 本地素材库

管理本地创作素材，包括图片、视频、音频等媒体文件。

## Scope

- 浏览和搜索本地素材文件
- 读取图片、视频、音频元信息
- 组织和分类素材文件
- 快速查找和引用素材

## Guidelines

### 推荐目录结构
```
素材库/
├── 图片/
│   ├── 背景/
│   ├── 图标/
│   └── 照片/
├── 视频/
│   ├── 片头/
│   ├── 转场/
│   └── 素材/
├── 音频/
│   ├── 背景音乐/
│   ├── 音效/
│   └── 配音/
└── 字体/
```

### 使用建议
1. 按类型和用途分类存放素材
2. 使用有意义的文件名便于搜索
3. 定期清理不再使用的素材
4. 重要素材做好备份

## Context

本地素材库用于管理创作过程中收集和制作的各类媒体文件。
配置路径可在启用后修改为你实际的素材存储位置。

支持的文件格式：
- 图片: jpg, png, gif, webp, svg
- 视频: mp4, mov, avi, webm
- 音频: mp3, wav, aac, flac
- 字体: ttf, otf, woff
