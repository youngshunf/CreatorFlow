# 智能办公工作区

你正在一个**智能办公工作区**中工作。这是一个专为办公文档处理设计的 AI 助手环境，支持 Word、PPT、Excel、PDF 等常用办公软件的文档创建、编辑和数据处理。

## 工作区特点

- **多格式支持**：Word (.docx)、PowerPoint (.pptx)、Excel (.xlsx)、PDF
- **完整操作**：创建、编辑、分析、转换
- **专业技能**：内置 4 个核心文档处理技能

## 目录结构

```
文档/
├── Word/      # Word 文档
├── PPT/       # 演示文稿
├── Excel/     # 电子表格
└── PDF/       # PDF 文档
模板/          # 文档模板
输出/          # 处理后的输出文件
```

## 内置技能

### 1. Word 文档处理 (docx)
- **创建文档**：从零开始创建专业 Word 文档
- **编辑文档**：修改现有文档内容和格式
- **文档分析**：提取文本、分析结构
- **批量修订**：支持追踪修改和批注
- **格式转换**：与其他格式互转

### 2. PPT 演示文稿 (pptx)
- **创建演示**：根据主题生成专业 PPT
- **模板应用**：使用现有模板快速制作
- **内容编辑**：修改幻灯片内容和布局
- **缩略图生成**：快速预览所有幻灯片
- **演讲者备注**：添加和编辑备注

### 3. Excel 表格处理 (xlsx)
- **数据分析**：使用 pandas 进行数据处理
- **公式计算**：创建和编辑复杂公式
- **数据清洗**：去重、格式化、异常值处理
- **图表生成**：创建数据可视化图表
- **财务模型**：支持专业财务建模规范

### 4. PDF 文档处理 (pdf)
- **文本提取**：从 PDF 中提取文字和表格
- **文档合并**：合并多个 PDF 文件
- **页面操作**：拆分、旋转、重排页面
- **表单填写**：填写 PDF 表单
- **格式转换**：PDF 与其他格式互转

## 常见工作场景

### 报告生成
1. 收集数据到 Excel
2. 分析数据生成图表
3. 创建 Word 报告文档
4. 导出为 PDF 分享

### 演示制作
1. 整理内容大纲
2. 选择合适的模板
3. 生成 PPT 演示文稿
4. 优化排版和设计

### 数据分析
1. 导入 Excel 或 CSV 数据
2. 清洗和预处理数据
3. 创建公式和透视表
4. 生成分析报告

### 文档转换
1. PDF 转 Word 进行编辑
2. Word 转 PDF 进行分享
3. PPT 导出为图片
4. 批量格式转换

## 工作原则

### 文档处理
- **保持格式**：编辑时尽量保留原有格式和样式
- **备份原件**：重要操作前备份原始文件
- **验证结果**：处理后检查文档是否正确

### 数据安全
- **敏感信息**：注意保护文档中的敏感数据
- **版本管理**：重要文档保留多个版本
- **错误处理**：遇到错误时提供清晰的解决方案

### 专业规范
- **Excel 公式**：使用公式而非硬编码值，保持动态更新
- **颜色规范**：财务模型遵循行业颜色标准（蓝色输入、黑色公式）
- **文档结构**：保持清晰的目录和章节结构

## 提示

- 处理大文件时会自动分批处理
- 支持批量操作多个文件
- 复杂操作会分步骤执行并报告进度
- 遇到格式问题会尝试自动修复
