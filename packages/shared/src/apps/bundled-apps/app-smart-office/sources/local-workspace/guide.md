# 本地工作目录

管理工作文档、报告、会议记录和办公资料。

## Scope

- 读取和搜索工作文档
- 整理和分类办公资料
- 提取会议纪要和任务
- 生成工作报告

## Guidelines

### 推荐目录结构
```
Work/
├── 项目/
│   └── {项目名}/
│       ├── 文档/
│       ├── 会议/
│       └── 资料/
├── 报告/
│   ├── 周报/
│   ├── 月报/
│   └── 年度/
├── 会议/
├── 模板/
└── 归档/
```

### 文件格式
- 文档: doc, docx, pdf, md
- 表格: xls, xlsx, csv
- 演示: ppt, pptx
- 图片: png, jpg, svg

### 使用建议
1. 按项目和日期组织文件
2. 使用统一的命名规范
3. 定期备份重要文档
4. 及时归档已完成项目

## Context

本地工作目录用于管理日常办公文档和资料。
启用后可修改路径指向你的工作文件夹。
