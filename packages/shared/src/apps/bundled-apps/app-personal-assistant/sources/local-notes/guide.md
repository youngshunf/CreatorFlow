# 本地笔记

管理个人笔记、备忘录、日记和待办事项。

## Scope

- 读取和搜索笔记内容
- 创建和编辑笔记文件
- 组织和分类笔记
- 提取待办事项和提醒

## Guidelines

### 推荐目录结构
```
Notes/
├── 日记/
├── 工作/
├── 学习/
├── 项目/
│   └── {项目名}/
├── 待办/
└── 归档/
```

### 文件格式
- 推荐使用 Markdown (.md) 格式
- 支持纯文本 (.txt) 文件
- 日期命名格式: YYYY-MM-DD

### 使用建议
1. 为笔记添加有意义的标题
2. 使用标签或文件夹分类
3. 定期整理和归档旧笔记
4. 重要笔记做好备份

## Context

本地笔记源用于访问和管理你的个人笔记文件。
启用后可修改路径指向你的笔记存储位置。

支持的笔记应用目录：
- Apple Notes 导出目录
- Obsidian vault
- Notion 导出
- 普通文件夹
