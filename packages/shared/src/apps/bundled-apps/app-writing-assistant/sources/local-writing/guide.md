# 本地写作目录

管理文章草稿、创作素材、写作笔记和参考资料。

## Scope

- 读取和搜索文章内容
- 管理草稿和已发布文章
- 组织写作素材和灵感
- 维护参考资料库

## Guidelines

### 推荐目录结构
```
Writing/
├── 草稿/
│   ├── 进行中/
│   └── 待修改/
├── 已发布/
│   ├── {年份}/
│   └── {平台}/
├── 素材/
│   ├── 灵感/
│   ├── 引用/
│   └── 图片/
├── 参考/
│   ├── 书籍笔记/
│   ├── 文章收藏/
│   └── 研究资料/
└── 模板/
```

### 文件格式
- 推荐使用 Markdown (.md) 格式
- 支持富文本 (.rtf, .docx)
- 纯文本 (.txt) 用于快速记录

### 使用建议
1. 草稿按进度状态分类
2. 已发布按时间或平台归档
3. 保留写作过程的版本
4. 定期整理和归档素材

## Context

本地写作目录用于管理你的写作项目和素材。
启用后可修改路径指向你的写作文件夹。

支持的写作工具：
- Markdown 编辑器
- Typora / Obsidian
- Ulysses 导出
- 普通文档编辑器
